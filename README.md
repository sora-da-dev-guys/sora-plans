# sora-plans 🌸

Centralized implementation plans for Sora Da Dev Guys projects, managed by Minna AI.

## Plan ID Format

`SORA-YY<seq>` — e.g. `SORA-26001`
- `YY` = 2-digit year
- `<seq>` = 3-digit sequential number (global across all repos)

## Status Lifecycle

```
draft → ready → working → reviewing → done
```
