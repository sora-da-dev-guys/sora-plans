### API function to add in `src/apis/sbb-api/apis.ts`

```ts
// GET /quotations/table-data
export const getQuotationTableDataApi = async (
  query: GetQuotationTableDataQuery
): Promise<IQuotationTableData> => {
  const queryString = buildQueryString(query as Record<string, string | undefined>)
  const response = await fetch(`${SBB_API_URL}/quotations/table-data${queryString}`, {
    method: 'GET',
    headers: getAuthHeaders(),
  })
  if (!response.ok) {
    if (response.status === 401) handleUnauthorized()
    const error = await response.json().catch(() => ({}))
    throw new Error(
      (error as { message?: string }).message || 'Failed to fetch quotation table data'
    )
  }
  return response.json()
}
```

### Hook: `src/hooks/sbb-api/useQuotations.ts`

```ts
import { useQuery } from '@tanstack/react-query'
import { getQuotationTableDataApi } from '@/apis/sbb-api'
import type { GetQuotationTableDataQuery } from '@/apis/sbb-api'

export const useQuotationTableData = (query: GetQuotationTableDataQuery) => {
  return useQuery({
    queryKey: ['quotations', 'table-data', query],
    queryFn: () => getQuotationTableDataApi(query),
  })
}
```

### Page: `src/pages/quotations/QuotationListPage.tsx`

**Columns (based on SBB spec):**

| Column | Source | Notes |
|--------|--------|-------|
| Status | `status` | Chip with colour by status id |
| Sale Order | `activeSaleOrderAmount` | Green doc icon if > 0 (with tooltip count), disabled icon if 0 |
| Quotation Code | `code` | Link to `/quotations/:id` |
| Created Date | `recCreatedAt` | Formatted `DD/MM/YYYY HH:mm` |
| Created By | `recCreatedBy` | Plain text |
| Customer Name | `customer.nameLocal` | Plain text |
| Phone | `customer.phoneNumber` | Plain text |
| Payment Method | `paymentMethod.name` | Plain text |
| Total | `total` | Formatted number `฿1,234.00` |

**Filters (via URL params):**
- Date range: `startDate` / `endDate` (filter on `recCreatedAt`)
- Status multi-select (derived from loaded data — distinct statuses)
- Checkbox: "No sale order yet" → `isNoSaleOrder=true`
- Checkbox: "Need to follow" → `isNeedToFollow=true`

**Pagination:** same URL-param pattern as `CustomerListPage` (`page`, prev/next, go-to).

**Status chip colours:**
```ts
const QUOTATION_STATUS_COLOR: Record<number, string> = {
```
