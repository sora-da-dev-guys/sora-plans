### Product type tabs

Reference grade values from SBB `productGrades` constants. Use these tab options:

```ts
const PRODUCT_TYPE_TABS = [
  { label: 'Product',          value: PRODUCT_TYPE.Product },
  { label: 'Service',          value: PRODUCT_TYPE.Service },
  { label: 'Internal Product', value: PRODUCT_TYPE.InternalProduct },
  { label: 'Compound Product', value: PRODUCT_TYPE.CompoundProduct },
  { label: 'Asset',            value: PRODUCT_TYPE.Asset },
]
```

Tab value drives the `typeIds` query param. Reset to page 0 on tab change.
